Bitwise

Conceptual Problems

1. Explain the different types of bitwise operators in C and describe how they work.

2. print the output of the following code snippet.

1 
int a = 10, b = 500;
for (int i=0; i<5; i++)
{
    printf("%d, %d\n", a, b);
    a <<= 1;
    b >>= 1;
};
2 
int a = 10, b = 11, c = 0;
printf("%d, %d\n", a | b, a & b);
printf("%d\n", (a || b) | (a && c));
3 
int a = 0x1f, b = 0xf1;
printf("%d, %d\n", a & b, a | b);
printf("%d, %d\n", a >> 6, b >> 6);
printf("%d, %d\n", a >> 4, b >> 4, (a >> 4) << 4, (b >> 4) << 4);
printf("%d, %d\n", a << 4, b << 4, (a << 4) >> 4, (b << 4) >> 4);
4 
int a = -0x1f, b = -0xf1;
printf("%d, %d\n", a & b, a | b);
printf("%d, %d\n", a >> 6, b >> 6);
printf("%d, %d\n", a >> 4, b >> 4, (a >> 4) << 4, (b >> 4) << 4);
printf("%d, %d\n", a << 4, b << 4, (a << 4) >> 4, (b << 4) >> 4);
5 
unsigned char c = 0xf1;
printf("%c, %u\n", ~c, ~c);
6 
int a = (1 << 4), b = (1 << 4) - 1;
printf("%d, %d\n", a, b);

Problems

1. write a program to print the binary representation of a given number.

2. Create a program that takes decimal number, a start index, and an end index as inputs. The program should then print the binary representation of the bits between the specified start and end indexes(inclusive).

3. Write a C program to set and reset specific bits of a given integer number. The program should take the following inputs:

a. the decimal number,

b. the position of the bit to set and

c. the position of the bit to reset

after performing the operation, the program should display the number and bit pattern of the modified bits.

4. Write a program that takes a decimal number, a starting index, and fetches the 4 bits starting from the specified index. Display the decimal value represented by the fetched 4 bits.

5. Create a program that takes a decimal number, a starting index, and sets the 4 bits starting from the specified index. Display the bit pattern of modified decimal number.

6. Create a program that takes a decimal number, a starting index, and resets(set to 0) the 4 bits starting from the specified index. Display the bit pattern of modified decimal number.

7. Write a c program that takes a decimal number, a start index and an end index. The program should set the bits between the specified start and end indexes(inclusive) in the decimal number and print the updated bit pattern.

8. Write a c program that takes a decimal number, a start index and an end index. The program should reset the bits between the specified start and end indexes(inclusive) in the decimal number and print the updated bit pattern.

9. Write a C program that takes a decimal number and a bit position as inputs. The program should toggle(flip) the value of the bit at the specified position in the binary representation of the decimal number. Display the modified bit pattern.

10. You are given two integers, 'source' and 'destination'. Your task is to replace the bits of the 'destination' integer, starting from a specified index, with the corresponding bits from the 'source' integer. Write a C program that takes the following inputs.

- 'source': The integer from which bits will be picked.

- 'source_start_index': The index where the bit picking should start.

- 'destination': The integer where selected bits from the source integer will be placed.

- 'dest_start_index': The index in the destination integer where bits will be replaced.

- 'num_bits': The number of bits to pick and replace.

11. Write a C program that an integer as input and counts the number of set(1) and reset(0) bits in its binary representation. Display bit battern and the count of set and reset bits.
